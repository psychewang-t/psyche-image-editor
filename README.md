### 项目作用

image-editor 是一个整合目前流行 AI 能力的在线版本图文编辑器，针对图片文字的基础操作能力都已经包含，支持导出 PNG、JPG 格式图片。
[在线体验地址](https://image-editor-demo-1ex4iya7dd3fb3-1257477541.tcloudbaseapp.com/ '在线体验地址')

### 如何运行此项目

1. 下载项目到本地
2. npm install 下载依赖包
3. npm run dev 运行
